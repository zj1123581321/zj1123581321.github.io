谈谈我的 2019 和 2020

2020-01-16 · 10 min read
总结

1 说说废话

本来脑海里构思 2019 总结的时候有一大堆话想说，但真正要写下来的时候，又不知道从何说起。

算了，那还是先从公众号开始吧，很抱歉 2019 年最后几个月公众号基本没更新。原因其实就两个，第一确实是忙，下半年很多事情赶到一块了，抽不出时间写公众号。第二自然就是懒了，把图文以公众号的形式表达出来，心理上还是有一定的压力的。

多数时候我都是在 Telegram 的私人频道里面发言的，一来 TG 全平台客户端比较方便，用手机可以直接分享一些内容；二来是 TG 相对自由，也不担心莫名敏感词无法通过。不过新的一年，我争取保证公众号每月至少一篇内容输出，以此倒逼自己输入。

2019 年个人在形态上没什么变化，但是在认知上，整个人上半年和下半年可以说是天壤之别。

大概是在 2019 的上半年，整个人还是偏理想主义的，成天关注一些 AI、区块链、这些被炒得很热的概念，每天感觉听了很多很多对对行业趋势的预测，对未来的憧憬，看上去仿佛获得了很多知识，但是一深究，就会觉得很空洞，说得都是一些大而无用的东西。

有些内容，比如说有的行业趋势分析，你不能说他不对，或许对于大的企业而言，它们有一些指引方向的作用。但是对于我这样微小的个体而言，它们有一个非常致命的问题——没办法落地。那么其实这些内容对于我而言就是有用的废话。看看也就罢了，但不能过于重视。就好像我们每天出门查的都是当地天气，没人会去看世界平均气温怎么样，这对我们的影响太小了。能真正落地执行的，才是我这样微小的个体需要关注的问题。

可能也是发现了这个问题，下半年慢慢地整个人的信息源就偏向个体执行类和认知类的内容。在这个过程中，要特别感谢一些老师的帮助。

第一个要感谢的是森淼老师，执行力和认知都非常高，过去一年也有幸看到森淼老师是怎么一步一步把一个利用信息差套利的项目落实到位的，非常佩服。森淼老师每天会经常在知识星球里分享一些优质的摘录、牛人、思考。通过森淼老师，发现了很多在不同领域拥有深刻思考+实操经验的前辈，也在落地一项可以套利的小项目。下面是森淼老师的知识星球和写过的文章：

0成本套利，30分钟打造星球返佣导航站

一次饭否账号套利复盘

第二个要感谢的是曹大。曹老师江湖人称“互联网百晓生”，作为一名从 PC 时代走过来的老兵，十分清楚很多互联网内容幕后非常 Real 的东西。并且曹老师还愿意讲他的思考和经验分享出来。过去一年，从曹老师这里得到了许多做互联网内容、做人的建议，修正了自己某些错误的认知。真的非常感谢曹大愿意分享经验给年轻人，对于这个年纪的我而言，经验是最宝贵的财富。曹大的分享的的确确帮助我在试错的路上少交了一些学费。

第三个要感谢的是西风老师。如果非要用一个词来形容西风老师，我想可能”人精“这个词最为合适吧。不论是什么事情，背后的利益纠葛、人性抉择都看得非常透亮。当然，有些事也因为说得太明白了让人有点难以接受，“记忆承载”堪称公众号里面的泥石流。

对事不对人？现实狠狠的扇了罗振宇一耳光！

其实还要感谢家人的支持、朋友的帮助，没有他们，或许我的 2019 年就是另一番风景了。

但是看了再多，这些内容于我而言也只能算是信息。未经思考、内化的难以称之为知识。所以下面说说 2019 我做了什么，以及 2020 我将要做什么。

2 做过的事和要做的事

2.1 做过的事

  1. 开通了公众号，上半年全渠道引流大概几千。对应知乎号，点赞 4k，收藏 2w。
  2. 年尾开始做视频内容，目前上传了一个视频，全网播放量 2w。
  3. 失物招领小程序基本制作完成。

坦白讲，还是看得多，做得少。可能在具体的技能点上有一定提升，但是相对于吸收的信息量而言，转化率很低。所以新的一年就只有一个主题——试错。

2.2 2020 任务

  1. 上线另一个小程序，日活做到 1000，能收支平衡。如果条件允许的话，尝试线上线下配合运作。
  2. 孵化一个月销千单的一次性业务。
  3. 全年全网播放量破 100w，b 站粉丝达到 1000，YouTube 播放量过 1w。
  4. 托福过 100...
  5. 瘦到 130...

如果顺利的话，小程序和线下业务的结合将会是 2020 工作的重心，或许未必能盈利，但获得了经验也不亏。第二项任务是给自己留个后路，不至于活得太窘迫。第三项任务则是趁着 B 站的红利期给自己占个坑，过两天我也会写一篇对于当下做视频内容的一点思考和建议，毕竟早就是优势，要抓住。最后两项，不能再拖了。

如果任务能够顺利完成，就给读者朋友们发个大红包。

2.3 长线任务

  1. 学习 SEO，孵化一个月 UV 过万的英文站。
  2. 提升自己的逻辑性，目前整个人还是缺乏思辨能力，有的文章隐隐觉得不对，但又说不出来哪里有问题。缺乏逻辑还是蛮致命的一个问题。
  3. 提升自己讲故事的能力。现在自己插科打诨还行，但要是讲故事，再有趣的故事到我的嘴里也是索然无味。

暂时就开这么多坑，慢慢填就是了。选择这些事情来做的原因也很简单：它们要么能够为我增长经验，要么可以一次投入，终身收获。毕竟，最大的套利机会就是时间。

3 2019 杂言

下面谈谈在 2019 年常常在脑海里回荡的内容、数据、感悟。

当一场浪潮到来的时候，最好的做法不是投身浪潮，而是为浪潮提供工具。

考研、考公、考证的有没有实现他们的理想我不知道，但我知道的是从源头的课程生产、到线上辅导、甚至是最终端二次卖课的，不少人过得挺滋润的。

公众号赚没赚到钱不清楚，但是为创作者提供技术支持的服务商们过得还不错。

在中国再众所周知的事情，都有一个亿的人不知道，但往往是十个亿的人都不知道。

信息差永远存在，信息差就是金钱。拿购物随便出两个问题，诸位可以测试一下，看看自己知不知道。

  1. 用过 1688 么？
  2. 为什么会有专门的网站、App、公众号给你发淘宝商品的优惠券？这些优惠券是哪里来的？

2019年会是过去十年中最差的一年，未来十年中最好的一年

目测会有一大批看上去能够解决焦虑的服务出来收割。扶住把手，小心别翻车了。赚想要赚钱的人的钱或许是最容易的：）

相对于西瓜视频，我更认可 B 站最有可能成为中国的 YouTube。趁着目前 B 站还需要扩充他的长尾内容，找准领域尽快切入，早就是优势。

4 榜单

4.1 2019 买过最值的电子产品

19 年初入手了一台二手 iPad 2018 32G，在基本解决了文件处理的问题之后，我的 Surface Pro Pen 就基本上退休了。硬件上延迟、偏移不如 iPad，软件上没有 iPad 的生态，除了屏幕素质好一些，Surface Pro ，唉。

通过 iPad+全面的软件生态 ，基本实现了无纸化生活。后来拼多多百亿补贴的时候还劝了身边的几个朋友入手 iPad，给他们写了简单的 iPad 文件管理入门（过两天润色一下发出来）。

对于经常需要书写的人而言，iPad 就一个字，真香。

4.2 2019 最看好的公司

拼多多、B 站、美团。

4.3 2019 最有价值的支出

虽然自己极度贫穷，但是还是为祖国的绿化事业尽了一份力的😀。

4.4 2019 最想推荐的书

2019 读书不多，读的大多目的性也比较强。不如给大家推荐一本非功利性的小说——《亮剑》，没错，就是那个李云龙的亮剑。小说大概读了三四遍，明明知道结局，可每次读完还是流泪。电视剧只是小说的前半部分，后面的内容或许更值得深思。

你可以在“微信读书”App 里用无限卡免费阅读，也可以购买实体书。

就说这么多吧。
