二 Windows 平台图床工具选择

2.1 Mpic

Windows 平台下应该是最好的图床工具了。有如下的优点。

  • 多种格式选择
  • 截图/复制图片自动上传
  • 链接自动复制到剪贴板
  • 图片自动瘦身
  • ...

缺点是只支持七牛云。

2.2 PicGo

新晋的 Windows、Mac、Linux 全平台图床工具。支持微博图床、七牛图床、腾讯云 COS v4/v5、又拍云、GitHub、SM.MS、阿里云 OSS、Imgur。

另外支持插件

官方使用说明，关于 PicGo 的更多介绍，你可以查看这里

三 Mac 平台图床工具选择

3.1 iPic

Mac 上的图床神器，目前支持微博（即默认图床）、七牛云 、又拍云、阿里云 OSS 、腾讯云 COS 、Imgur 、Flickr 、Amazon S3

支持复制上传、手动上传、拖曳上传。

作者的开发历程，更多介绍

同时还有 iPic Mover，可以通过 iPic 上传 Markdown 中的图片（包括本地或网络图片）至新的图床，并替换 Markdown 中的图片地址

3.2 PicGo

见上文。

四 Chrome 平台图床工具选择

4.1 七牛在线存图

新浪图床挂了之后，Chrome 上很难找到像微博图床这样强大的插件了。对我而言，Chrome 平台图床插件的一个刚需是直接上传网页图片到图床，我找到了「七牛在线存图」这个插件勉强替代。不过只支持七牛云、生成的链接不能直接复制到剪贴板、不能直接传 Gif...只能是将就一下了。
