一 系统优化

1.1 软件卸载

新买的电脑或者重装的系统可能会有一些预装软件，如果对你而言软件无用的话，建议右键直接卸载。

列几个常见的预装软件

  • Microsoft Store 里面的某些游戏

  • McAfee迈克菲全方位实时保护

  • 纸牌游戏

  • XBOX(如果不玩游戏的话)

    如果不知道一个软件有什么用，请自行复制查询，或者干脆不管它。

    关于常见品牌的预装软件及其作用，可以参考这篇文章

1.2 自启动管理

1.2.1 管理自启动

同时按住 Ctrl+Shift+Esc （都在你键盘左侧）调出任务管理器，左键单击『启动』

在你需要禁止的项目上右键调出菜单即可禁用该自启动项目。

这种方式应该是管理自启动最自然的方式。

如果有需要的话，下文也会推荐一个小软件进行高级管理。

1.2.2 添加自启动

同时按住 Windows+R 键，调出『运行』窗口 ，输入 “shell:startup” （注意冒号是英文冒号）后回车。

如果想要让某个软件自启动（假设软件自身无法进行设置），将该软件的快捷方式放到这个文件夹里面一份就好。

一般的软件都会在安装时创建桌面快捷方式，你可以选择将这个快捷方式复制进去。更多的方法因为涉及到软件安装位置的问题，这里就不赘述了，请自行百度。

1.3 主题优化

这里我个人不太喜欢 Windows 原来的一套古老图标，所以找了一份顺眼的图标进行替换 （Win10 1909 以下亲测好用）。非必须，看个人喜好。

替换主题时操作略复杂，请提前创建系统还原点，具体操作指南和注意事项请参考官方指北。

1.4 文件管理

1.4.1 图片、视频、等文件夹迁移位置

Win10 自带的七大类别文件夹默认是在 C 盘的，如果你 C 盘只有 128G 的话，建议你修改他们的默认位置。

方法：在要修改的文件夹上依次点击“右键”-“属性”-“位置”-“移动”

1.4.2 OneDrive 设定位置

云盘应该是当下使用电脑的一个标配。文件同步/备份都离不开一份安全可靠方便的云服务。

OneDrive 是微软亲儿子，和 Win10 的结合比较紧密。普通用户有 5G 的空间可用，Office 365 用户有 1T 空间可用。建议重要文件及时放到 OneDrive 或者是其他云盘里备份/同步。

这里主要是提醒诸位第一次设置 OneDrive 请谨慎选择默认位置，之后无法修改。C 盘是 128G 的朋友就最好别放在 C 盘了，或者及时开启 『按需同步』的功能减少本地空间占用。

1.4.3 安装 Clover ，常用文件夹固定成目录

Clover 主要是可以让文件管理器拥有多个标签页和目录，简化了文件管理。

QTTabBar 也能实现多标签页管理，甚至某些功能更强大，但是缺少 Clover 这样一步到位的目录栏。结合自身需求选择就好。

这里特别提醒，Clover 官网新版本已经有很多弹窗和推广内容，非常不建议安装。低版本也会要求强制升级。

建议禁止 Clover 联网并且禁止其启动自动更新服务。

方法：

首先需要禁用 Clover 联网，具体可参见 Win 10禁用某个程序联网的教程。

之后 Win+R 调出『运行』面板，输入 services.msc，找到 HcloverService，“右键”-“属性”-“禁用”

或者安装网友的去广告优化版，删除了升级模块还有这些推广模块。

Win10 会报毒并自动删除，可通过设置排除项解决。

1.5 右键菜单管理/新建文件管理

警告：清理右键菜单之前请务必做好注册表备份，避免出现灾难性后果！

其实这部分放在软件安装完做最好，那个时候多半你的右键菜单栏已经被填充得密密麻麻了...

这方面我体验过市面上很多款右键管理清理工具，火绒、RightMenuMgr、还有各种小工具，说实话，做得都比较浅，一些右键菜单根本无法清理。

最终发现了 Right Click Enhancer 这款右键深度清理工具，确实是非常好用，完全能够满足管理右键菜单的诸多需求，具体功能如图。

1.6 系统设置优化

1.6.1 创建还原点

创建系统还原点相当于给电脑拍了一张照片，日后可以根据这张照片恢复成这个还原点时电脑的相关配置、软件。系统还原点是 Windows 非常实用的一个功能。

建议装完系统做好基本配置并且安装好常用软件时手动创建一个还原点，方便出了问题时还原；同时也建议折腾电脑前创建一个还原点，给自己留一个退路，毕竟重装系统成本其实挺高的。

1.6.2 利用 Dism++ 进行系统优化

Dism++ 是一款非常强大的系统优化工具。像日常垃圾清理、备份更新、关闭一些无用的功能这些方法可能散落在 Windows 各处——注册表、组策略、服务..修改起来很不方便，Dism++ 把这些工具方法抽象出来，做成了一个工具箱真正做到了无门槛使用。

2.1 UWP 版软件

2.1.0 背景知识

UWP即 Windows 10 中的 Universal Windows Platform 简称。即 Windows 通用应用平台，在Windows 10 Mobile/Surface（Windows平板电脑）/PC/Xbox/HoloLens 等平台上运行，UWP 不同于传统 PC 上的 exe 应用，也跟只适用于手机端的 App 有本质区别。它并不是为某一个终端而设计，而是可以在所有 Windows10 设备上运行。

摘自：百度百科

用人话讲，UWP 软件就是你从 Microsoft Store 能下载的软件。它比传统的 exe 文件更安全、管理起来也更方便、更节省系统资源。但是生态做得不好，并不广为人所知。

即便如此，Microsoft Store 里面还是有不少精品应用的。

之所以推荐使用 UWP 软件，除了安装/更新方便、界面优化好之外，主要是里面的软件是和账号绑定的，下次重装系统可以直接从『我的软件库』里面快速重装。

2.2 绿色便携软件

2.2.0 背景知识

绿色便携软件你可以理解成那些无需你安装，即开即用的软件。更详细的定义你可以参考这里。

相较于需要安装的软件，它的优点在于重装系统/换电脑时只需要拷贝这些便携软件所在的文件夹即可，无需重新安装，极大减少了操作的繁琐性。

缺点就是现在常见软件一般不提供便携版，很多软件都是网友利用安装版本制作的。

小部分提供便携版的软件，安装包文件名中通常都包含 『Portable 』。

找软件的绿色便携版通常需要你具备一定的搜索能力和积累，比如 GitHub、精品绿色便携软件、PortableAppK。

而分布式网盘则是你自己就是服务器（或者说所有拥有这个文件的机器都是服务器），文件不经过云端直接传到对方电脑上面。隐私并且高速。

关于分布式网盘同步的更多资料你可以参考这篇文章：分布式网盘—Syncthing介绍与使用心得 +教程ver 8.3.2

  • 超过 3 个人的团队文件协作，每个人的电脑上都会安装进行文件协作。

也特别提醒

  • 分布式就意味着必须有至少两台设备同时在线才能进行同步

这款小工具只有一个功能——让某个窗口置于最前端显示。

用语言描述 Quicker 是无力的，直接上动图和视频。

三 日常使用

3.1 软件新装

3.1.0 下载

文章开头提过，这里再强调一遍。

  • 尽量去官网或者 GitHub 下载，遇上流氓的概率会小很多。
  • 建议如果空间足够的话，下载的安装包不要删除，直接移入“重装系统备份”文件夹。重装的时候能减少很多麻烦。
  • 小众软件、小工具记得做好安装包备份。
  • 大型软件最好本地留也留有备份，不然几个 G 的文件再从网上拉下来可能会很耗时。

3.1.1 安装

  • 所有国产软件，请不要选择『快速安装』、『一键安装』这类选项，请一步一步检查。
  • 安装位置最好可以统一管理，具体可以参考奶酪的这篇文章：大道至简，如何高效使用 Windows？。当然，懒得管理的话选择默认位置也没关系。
  • 安装过后记得去自启动管理检查一下，不要相信软件自身的自启动选项。
  • 可以利用开始菜单作为软件入口，为软件进行分类管理。

3.2 文件备份

电脑丢了，无非就是损失一点金钱，日后也能赚回来。但数据要是丢了，你花再多的钱也找不回来了。

再怎么重视数据安全也不为过，这方面展开讲也可以单开一个栏目，限于主题就简单提几嘴网络备份的选择。

普通 Windows 用户其实用好 OneDrive 做备份就足够了，5G 的空间不够的话可以拼车 Microsoft 365 家庭版，一年 50 多元，1T 空间可用。 当然，也可以直接购买 Microsoft 365 个人版，省去很多麻烦。

这里的 Microsoft 365 不光是 OneDrive 的 1T 空间，还包括 Word、Excel、PowerPoint 三件套的多平台使用权。

非常不建议你使用淘宝几块钱 5T 的 OneDrive 账号进行文件备份，这种账号是虚假注册的，随时可能被微软封号。而且账号里的文件可以被系统管理员随时查看和删除，毫无隐私性可言。

至于百度网盘，如果你是会员的话，Windows 倒也支持最多 5 个本地文件备份，这也不失为一种选择。

而坚果云，上面也提到过，如果有多设备同步的需求，它在国内会是首选。如果没有这方面的需求，也可以下载尝试一下，免费用户不限制空间、速度，只限制容量。（坚果云看到请给我续费打折，谢谢）

关于数据安全的更多知识，你可以参考这些文章：

3.3 系统安全

虽然大家都劝说微软自身的『Windows 安全中心』足够维持系统安全，没必要安装第三方安全软件。

但如果你对电脑了解不是很多，或者面对流氓弹窗束手无策，我还是建议你安装第三方安全软件。『Windows 安全中心』防得住病毒，防不住流氓。

不过我也不建议你安装流氓头子 36o 或者各路 XX 电脑管家/大师，安一个火绒就好。干净省心省事，也能够解决绝大多数流氓。

3.4 发现并解决工作流上的痛点

尽管上面推荐了 70+ 的软件，但是其实本文价值最高的部分藏在下面的几段话里面。

太阳底下无新鲜事，其实我们在操作电脑时遇到的问题里面 80% 前人都遇到过，并且可能给出了软件或者解决方案。

你需要做的其实就是：

  • 不断发现工作流中的痛点、可优化点
  • 以合适的方式进行搜索、提问
  • 找到/创造解决方案并记录（印象笔记剪藏或者打印成 PDF）

比如说，我觉得快捷方式图标里的小箭头很难看。我就会搜索“删除 快捷方式小箭头”，果然一大堆解决方案。

这样的例子还有很多，以终为始，让时间一点点打磨你的工作流，慢慢的，你的 PC 就会变成你的一件『利器』。
